🛡️ SYNK-F CyberMind

AI-Powered Cybersecurity Awareness Platform

📖 Overview

SYNK-F CyberMind is an AI-powered cybersecurity awareness platform designed to make cybersecurity learning simple, interactive, and accessible. The platform combines Generative AI with cybersecurity education to help users understand cyber threats, identify phishing attempts, and improve their knowledge through dynamic AI-generated quizzes.

✨ Features

- 🤖 AI Cyber Tutor – Ask cybersecurity questions and receive AI-generated explanations.
- 🎣 AI Phishing Analyzer – Analyze suspicious emails, SMS messages, and URLs.
- 📝 AI Quiz Generator – Generate unique cybersecurity quizzes using AI.
- 📊 Interactive Dashboard – Simple interface to access all core features.

🛠️ Built With

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Google Gemini API
- GitHub
- Vercel

🚀 How It Works

1. Open the application.
2. Log in to access the dashboard.
3. Choose one of the AI-powered features.
4. Learn cybersecurity through AI-generated responses, phishing analysis, and interactive quizzes.
